# Project Management

A Claude Code skill for managing non-coding projects (client work, personal projects, planning) with full scaffolding, Google Docs integration, and memory tracking.

## Documentation

See [SKILL.md](./SKILL.md) for complete documentation and usage instructions.

## Installation

To use this skill with Claude Code, clone it into your skills directory:

```bash
git clone https://github.com/HartreeWorks/claude-skill--project-management.git ~/.claude/skills/project-management
```

Then restart Claude Code to load the skill.
