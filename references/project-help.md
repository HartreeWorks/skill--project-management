# /project-help

Display all available project management skills with brief descriptions.

## Output

Display the following:

```
Project Management Skills
═══════════════════════════════════════════════════════════

/new-project      Create a new project with full scaffolding
/add-google-doc   Add a Google Doc to project
/create-doc       Create a new Google Doc for the project
/list-projects    Show all projects from central index
/project-context  Show current project orientation
/tasks            View and manage project tasks
/migrate-project  Import a project from Claude.ai
/archive-project  Archive a completed project
/project-help     Show this help

═══════════════════════════════════════════════════════════

Projects location: ~/Documents/Projects/
```

## Notes

- Most skills require being in a project folder (except /new-project, /list-projects, /migrate-project)
- Use /new-project to create your first project
- Use /list-projects to see all your projects
